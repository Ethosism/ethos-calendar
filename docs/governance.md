# Calendar Governance

The Ethos Calendar governs recurrence, not doctrine.

It should remain subordinate to the Ethos canon, compatible with Ethra, and accountable to real-world consequences.

## Change Status

Calendar practices use these statuses:

- Draft: private or early work.
- Proposed: ready for review.
- Experimental: usable in bounded trials with review criteria.
- Accepted: incorporated into the calendar source of truth.
- Deprecated: retained for history but no longer recommended.
- Rejected: reviewed and declined with reasons.

## Admission Rule

A calendar practice should be accepted only if it:

1. Makes reality easier to see.
2. Clarifies duty.
3. Reduces avoidable harm.
4. Improves repair.
5. Preserves memory.
6. Resists coercion, personality cults, and vague self-help drift.
7. Can be performed in plain language without Ethra fluency.

## Boundaries

Calendar practices must not:

- require confession to a facilitator or group
- create spiritual ranks
- sell authority
- override professional, medical, legal, or emergency processes
- shame people for limits such as illness, poverty, grief, disability, age, or dependency
- turn private reflection into surveillance

## Proposal Shape

A proposed practice should include:

- name
- status
- purpose
- recurrence
- plain-language instructions
- affected Ethos principles
- optional Ethra hinges
- risks and misuse cases
- boundaries
- review method

## Review Method

Review should ask what the practice actually produced.

Good evidence includes:

- clearer truth
- completed repair
- improved stewardship
- stronger memory
- reduced avoidance
- better boundaries
- more reliable service

Sentiment alone is not enough.
