# Life Rites

Life rites mark transitions where truth, duty, memory, and relationship become visible.

These rites are not sacraments and do not require clergy. They are structured practices that help people tell the truth about a transition and name what follows from it.

## Naming

Purpose: welcome a person into memory, care, truth, and future responsibility.

Core acts:

- Say the name.
- Name the people responsible for care.
- Record what is being promised and by whom.
- Bless without claiming ownership of the person's future.

## Coming Of Age

Purpose: mark increased agency, duty, skill, and accountability.

Core acts:

- Name new freedoms.
- Name new duties.
- Name skills already learned.
- Name skills still required.
- Invite elders or mentors to give concrete counsel.

## Vocation

Purpose: name useful work and the responsibilities carried by craft.

Core acts:

- Describe the work plainly.
- Name who is affected by the work.
- Name standards of honesty, competence, and repair.
- Record one commitment to improve the craft.

## Marriage

Purpose: bind public promise, private love, household duty, and future consequence.

Core acts:

- State the promise plainly.
- Name the household duties being accepted.
- Name how conflict and repair will be handled.
- Invite witnesses to support the marriage without controlling it.

## Reconciliation

Purpose: mark truth-telling, accountability, repair, and restored or clarified relationship.

Core acts:

- State what happened without evasion.
- Name harm and responsibility.
- Name repair already made and repair still owed.
- State whether the relationship is restored, bounded, or released.

## Grief

Purpose: tell the truth about loss and preserve memory without denying pain.

Core acts:

- Name what has been lost.
- Tell one true memory.
- Name what remains unfinished.
- Name what care is needed now.

## Elder Honor

Purpose: recognize age, contribution, dependence, memory, and unfinished transmission.

Core acts:

- Ask what should be remembered.
- Name contributions truthfully, without exaggeration.
- Name present needs without shame.
- Record stories, instructions, or warnings worth carrying forward.

## Death Remembrance

Purpose: remember the dead and examine what remains entrusted to the living.

Core acts:

- Speak the name.
- Tell the truth about the life.
- Preserve memory.
- Name duties that pass to the living.
